# Portfolio

A responsive, single-page portfolio you can deploy anywhere.

## Quick start
- Open `index.html` in a browser, or serve the folder locally:
  - Python: `python3 -m http.server 8000`
  - Node: `npx serve`

## Customize
- Replace "Your Name" and content in `index.html` (Hero, About, Experience).
- Update links for GitHub/LinkedIn/Email in the footer.
- Edit `projects` array in `script.js` to populate project cards.
- Tweak colors and spacing in `styles.css` (`:root` variables).

## Deploy
- GitHub Pages: push to a repo, enable Pages, set branch to `main` and folder to `/`.
- Netlify/Vercel: import the repo and deploy as a static site.

## Assets
- Hero image uses Unsplash; replace with your own image file or link.
